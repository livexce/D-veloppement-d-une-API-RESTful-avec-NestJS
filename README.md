# NestJS MongoDB API

Ce projet est une API RESTful développée avec NestJS et MongoDB.

## Prérequis

- [Node.js](https://nodejs.org/) installé sur votre machine
- [MongoDB](https://www.mongodb.com/try/download/community) installé localement ou accessible en ligne

## Installation

1. Clonez ce dépôt :

```bash
git clone https://github.com/votre-utilisateur/votre-repo.git

1-Accédez au répertoire du projet :
cd nest-mongo-api
2-Installez les dépendances :
npm install
3-Démarrez l'application :
npm run start
L'API sera accessible à l'adresse http://localhost:3000.
